# 🚀 Micro-Post Generator

> Turn a seed thought into something you can actually post.

---

## The Formula

```
[Relatable struggle] + [What I tried] + [Small insight] + [One-line takeaway]
```

Fill in the blanks. Post it. Done.

---

## 5 Ready-to-Use Templates

### Template 1: The Lesson
```
I used to [old behavior].

Then I tried [new thing].

Turns out [small insight].

The lesson: [one sentence].
```

### Template 2: The Confession
```
Honest take: I [struggle/flaw].

Here's what actually helped: [simple thing].

Not perfect. But good enough to ship.
```

### Template 3: The Tiny Discovery
```
Small thing I noticed: [observation].

Why it matters: [one reason].

Try it: [one actionable step].
```

### Template 4: The Reframe
```
[Common belief everyone holds].

But what if [reframe]?

[One example that proves it].
```

### Template 5: The ML Dev Edition *(bonus for devs)*
```
I trained a model to [task].

The hardest part wasn't the code — it was [real challenge].

What I'd tell myself at the start: [insight].
```

---

## How to Use This

1. Pick a seed from your daily note
2. Choose the template that fits
3. Fill in the blanks (don't overthink it)
4. Post it as-is, or lightly edit

**Target length:** 50–150 words. Shorter is better.

---

## The Anti-Perfectionism Rule

> A posted rough draft beats a perfect post that never ships.

If you're editing for more than 5 minutes, post it anyway.
